3. Cài đặt và Kiểm thử phần mềm

Hệ thống SLTMS v1.0

3.1. Hiện thực hóa hệ thống

🛠️ Công nghệ chủ chốt

  • Backend: .NET 8.0 & C#
  • Database: SQL Server 2022 & EF Core
  • Hàng chờ & Cache: RabbitMQ & Redis
  • Kiến trúc: Layered Architecture (4 lớp tách biệt)
  • Bảo mật: JWT (JSON Web Token) phân quyền Role-based
Triển khai Infrastructure (Docker): docker run -d --name rabbitmq -p 5672:5672 rabbitmq:3-management

📊 Môi trường Kiểm thử (Testing)

Phần cứng:

  • CPU Intel Core (Iris Xe)
  • RAM 16GB DDR4 3200MT/s
  • SSD NVMe High Speed

Công cụ:

  • Apache JMeter (Non-GUI)
  • Postman (API Test)

🎯 Mục tiêu Kiểm thử

  • Unit Test: Kiểm tra logic nghiệp vụ đơn lẻ.
  • Load Test: Đánh giá độ ổn định khi tải cao.
  • Xác thực: Minh chứng đáp ứng yêu cầu bài toán.
